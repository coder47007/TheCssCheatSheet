CSS Cheat Sheet

A comprehensive CSS cheat sheet with examples, properties, and techniques.

Introduction to CSS

What is CSS? CSS (Cascading Style Sheets) is used to style the presentation of web pages, including layout, colors, and fonts.

CSS Syntax:

selector {
    property: value;
}

Example:

h1 {
    color: blue;
    font-size: 24px;
}

CSS Selectors

Element Selector

p {
    color: green;
}

Class Selector

.class-name {
    font-size: 18px;
}

ID Selector

#id-name {
    color: red;
}

Universal Selector

* {
    margin: 0;
    padding: 0;
}

The CSS Box Model

The box model describes the rectangular boxes generated for elements in the document tree, consisting of content, padding, border, and margin.

.box {
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 2px solid #000;
    margin: 15px;
}

CSS Backgrounds

Background properties define the background color, image, size, and more.

Background Color

body {
    background-color: #f4f4f4;
}

Background Image

.hero {
    background-image: url('hero.jpg');
    background-size: cover;
    background-position: center;
}

Text Styling

Font Properties

p {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: bold;
}

Text Align

h1 {
    text-align: center;
}

Text Transform

h2 {
    text-transform: uppercase;
}

Pseudo-Classes

Pseudo-classes define the special state of an element.

Hover

a:hover {
    color: red;
}

Focus

input:focus {
    border-color: blue;
}

First Child

p:first-child {
    font-weight: bold;
}

Borders

Border Properties

.box {
    border: 1px solid #ccc;
    border-radius: 5px;
}

Layout Techniques

Flexbox

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

CSS Colors

Named Colors

p {
    color: aqua;
}

RGBA Colors

div {
    background-color: rgba(0, 0, 255, 0.5);
}

Creating Shapes

Circle

.circle {
    width: 100px;
    height: 100px;
    background: red;
    border-radius: 50%;
}

Triangle

.triangle {
    width: 0;
    height: 0;
    border-left: 50px solid transparent;
    border-right: 50px solid transparent;
    border-bottom: 100px solid blue;
}

Transitions

button:hover {
    transform: scale(1.1);
    transition: transform 0.3s;
}

CSS Clip-Path

.clip {
    clip-path: circle(50% at 50% 50%);
}

CSS Filters

Blur

img {
    filter: blur(5px);
}

Visibility

Control Element Visibility

.hidden-element {
    visibility: hidden;
}

Overflow

Handle Overflow Content

.content-box {
    overflow: auto;
}

Z-Index

Control Element Stacking

.overlay {
    z-index: 10;
}
